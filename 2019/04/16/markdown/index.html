换行和分段

  • 换行：只需在行末加两个空格键和一个回车键。快捷键：control + enter；
  • 分段：段落之间空一行即可；

文本样式

  • 加粗 - 快捷键： ctrl + B
  • 斜体 - 快捷键：command +
  • 删除线 - 快捷键：command + U
  • 底纹 - 快捷键：command + K
效果如下：
* 加粗：** 加粗 **
* 斜体： * 斜体 *
* 删除线：~~删除线~~
* 底纹： ` 底纹 `
